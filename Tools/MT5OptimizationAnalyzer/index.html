MT5 XML Analysis

Потоковый анализатор оптимизации для больших XML

Этот интерфейс работает через локальный Python backend и подходит для крупных отчетов MT5. Сервер читает Spreadsheet XML потоково, считает агрегаты и уже потом отдает браузеру компактный JSON.
